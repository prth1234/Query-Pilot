### 1. **Row Height** (3 options)
- **Compact**: Minimal padding for maximum data density

### 3. **Font Family** (3 options)

- **Serif**: Traditional reading font

### 4. **Header Style** (3 options)

- **Light**: Lighter header background

## Tips & Best Practices

### For Maximum Data Density:
- Row Height: **Compact**
- Font Size: **10-11px**
- Wrap Text: **OFF**
- Row Numbers: **OFF**

### For Presentations/Screen Sharing:
- Row Height: **Comfortable**
- Font Size: **16-18px**
- Font Family: **Sans Serif**
- All visual effects: **ON**

### For Analysis Work:
- Row Height: **Normal**
- Font Size: **13px**
- Font Family: **Monospace**
- Striped Rows: **ON**
- Sticky Header: **ON**

---

## Performance Considerations

Some settings impact performance with large datasets:

**Faster** (Better for large datasets):
- Wrap Text: OFF
- Simple borders
- Compact row height

**Slower** (Better for smaller datasets):
- Wrap Text: ON
- Complex header styles
- Comfortable row height

---

## Default Configuration

The default settings are optimized for:
- **Compact row height** - More data on screen
- **Monospace font** - Better data alignment
- **All visual aids ON** - Better UX
- **13px font size** - Balance between readability and density
